WLANalyse by raphaelm

A WLAN Statistic tool, written in PHP for Linux/Ubuntu

WLANalyse is a software which is looking for WLAN accesspoints in the background. It writes all founded accesspoints in a SQLite database file specified in config.inc.php. The sense of this program: After using it long time you can generate statistics about protected and unprotected accesspoints in your city. WLANalyse is able to generate nice and graphical HTML statistics with all the collected data.

Dependencies

- Linux/Unix - The command "iwlist" - WLAN - PHP5 (minimal ubuntu-packages: php5-cli, php5-sqlite for statistics php5-gd) - You need to be root.

Install

$ ./install.sh $ mv config.inc.example.php config.inc.php Customize the configuration file: $ vim config.inc.php Done!
